![Status do Projeto](https://img.shields.io/badge/status-em%20desenvolvimento-green)

Este é o meu portfólio profissional e acadêmico, desenvolvido para consolidar meus conhecimentos em **Engenharia de Software**. O projeto foca em uma interface limpa, funcional e responsiva, destacando minhas habilidades em Desenvolvimento Full-stack e Análise de Dados.

## Tecnologias e Ferramentas

Para a construção deste projeto, utilizei as seguintes tecnologias:

*   **Front-end:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)
*   **Ícones & Fontes:** Font Awesome, Google Fonts
*   **Versionamento & Deploy:** Git, GitHub, GitHub Pages

---

## Funcionalidades

- **Layout Responsivo:** Adaptado para diferentes resoluções de tela.
- **Interatividade com JS:** 
  - Botão de cópia de e-mail com feedback visual dinâmico.
  - Integração direta com API do WhatsApp para contato imediato.
- **Navegação Inteligente:** Menu fixo (sticky) com *Smooth Scroll* para melhor experiência do usuário.
- **Hierarquia Visual:** Uso de pseudo-elementos e ícones para facilitar a leitura técnica.

## Design & Inspiração

A identidade visual deste projeto foi cuidadosamente planejada para refletir um perfil técnico e moderno:

* **Layout:** Inspirado no template [Subtle](https://subtle.framer.website/) do Framer, adotando uma estética minimalista e focada em conteúdo.
* **Paleta de Cores:** Extraída de referências de Dark Mode de alto nível:
    * **Background:** `#0d0d0d`
    * **Cards/Header:** `#262626`
    * **Destaques:** `#333333`
* **Tipografia:** Utilização da fonte **Open Sans** via Google Fonts para garantir legibilidade e conforto visual.
